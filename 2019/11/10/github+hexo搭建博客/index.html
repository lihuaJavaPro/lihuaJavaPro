github+hexo搭建博客

（一）准备环境

准备 nodejs 和 git 环境，github账号

（1）nodejs

下载链接如下：Node.js下载 （下载安装提示：下载稳定版TLS；同时添加环境变量到电脑）

（2）git

ssh免密登录配置：参考链接

提示：公钥生成位置查看生成秘钥后的位置提示

（3）github

新建仓库（提示：仓库名必须是：用户名.github.io；Settins->GitHub Pages必须出现Your site is published at https://lihuajavapro.github.io/才算搭建github博客成功；域名可以save） 参考链接

（二）配置本地hexo

  • 安装Hexo(参考链接)，在自己认为合适的地方创个文件夹，我是在桌面上myblog文件夹。然后通过命令行进入到该文件夹里面

    • (1)输入npm install hexo -g，开始安装Hexo

    • (2)输入hexo -v，检查hexo是否安装成功

    • (3)输入hexo init，初始化该文件夹,最近Github好像有点问题，访问速度异常的慢，中间还失败了一次，所以这一步时间有点长，慢慢等。

    终于好了，看到后面的”Start blogging with Hexo！”激动啊有木有！

    • (4)输入npm install，安装所需要的组件

    • (5)输入hexo g，首次体验Hexo

    • (6)输入hexo s，开启服务器，访问该网址，正式体验Hexo

    • (7)在浏览器打开http://localhost:4000/ 出现如下图就成功了，体验Hexo，极简的风格，很棒。]
